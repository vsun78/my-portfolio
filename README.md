# Portfolio Website

Personal portfolio website for Victor Sun, a Software Engineering student at Western University.

## About

This portfolio showcases my professional experience, technical skills, and projects in software development and AI/ML engineering. The website features a modern, responsive design with smooth animations and interactive elements.

## Built With

- React 19.1.0
- JavaScript (ES6+)
- CSS3
- HTML5

## Key Libraries

- React Router for navigation
- GSAP for animations
- Font Awesome for icons
- Google Fonts (Inter)

## Structure

The portfolio includes the following sections:

- Hero section with introduction
- About Me with education details
- Work Experience timeline
- Technical Skills organized by category
- Featured Projects with descriptions
- Contact form and social links

## Development

This project was bootstrapped with Create React App and uses modern React features including hooks and functional components.
